PrintPure Privacy Policy

Last updated: March 23, 2026

PrintPure is a Chrome extension that creates a cleaner, print-friendly preview of the webpage you choose.

What PrintPure Accesses

When you click the extension on a webpage, PrintPure reads the current tab so it can extract the main article content and open a clean preview.

This means the extension handles:

  • the current page URL
  • the current page's readable article content

What PrintPure Stores

PrintPure stores only the minimum local data needed for the preview experience:

  • the current extracted article in chrome.storage.session
  • the source tab ID and preview tab ID in chrome.storage.session
  • the source page URL in chrome.storage.session
  • local display preferences such as theme and image visibility in browser local storage

Session storage is used so the preview can access the current extracted article during the active browser session.

What PrintPure Does Not Do

PrintPure does not:

  • create user accounts
  • collect personal information
  • sell user data
  • send article content to external servers
  • run analytics or tracking
  • use data for advertising

Third-Party Services

PrintPure does not use third-party analytics, advertising, or remote content-processing services in the current implementation.
